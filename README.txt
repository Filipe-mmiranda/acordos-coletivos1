
# ACORDOS COLETIVOS – PROJETO PRONTO PARA DEPLOY

## Como usar:

1. Acesse https://github.com/new e crie um repositório chamado `acordos-coletivos`
2. Faça upload de todos os arquivos desta pasta (exceto este .zip)
3. Acesse https://vercel.com/new
4. Conecte com o GitHub e selecione o repositório
5. Clique em Deploy — pronto!

## Sobre o projeto

- Feito em React (Next.js)
- Tema azul claro com texto preto em Arial 12
- Seleção por cidade (São Paulo, Bragança Paulista, São Carlos)
- Busca por cláusulas
- Exportação para TXT e PDF
